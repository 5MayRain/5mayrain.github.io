Windows 10 启用 Linux 子系统

Windows 10 启用 Linux 子系统

七月 06, 2021
  • Ubuntu
  • Windows
  • 教程

1. 启用开发人员模式

设置 > 更新和安全 > 开发者选项 > 启用开发人员模式

2. 更改系统功能

使用Win+X快捷键调出系统管理菜单(也可以右击Win开始菜单调出) > 应用和功能 > 程序和功能 > 启用或关闭 Windows 功能 > 勾选适用于 Linux 的 Windows 子系统 > 确定 > 重启电脑

3. 安装Linux子系统

重启系统后，打开 Microsoft Store，搜索 WSL，选择 Ubuntu 20.04 LTS，安装即可
