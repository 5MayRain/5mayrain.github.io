下载

选择 3dmigoto-GIMI-for-playing-mods.zip 下载

GI Model Importer

来源: GitHub


使用

编辑 d3dx.ini

解压下载的压缩包，打开 d3dx.ini 文件，搜索 loader ,找到 target =

找到你的游戏路径后，将 target = 后面的路径修改成你的游戏路径，然后保存即可

Mod 使用方法

打开 GameBanana 网站，找到你想要的 Mod

例如：我想要下图八重神子的 Mod

把她下载下来

打开下载的 Mod 压缩包，把里面的文件夹，解压至 3dmigoto/Mods 的文件夹内

启动

用管理员模式打开 3DMigoto Loader.exe

然后打开游戏，启动成功的话，命令窗口会消失，游戏内按 F10 即可刷新模型
